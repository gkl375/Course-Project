# Heap Data Structure and Heap Sort Algorithm

This repository contains a implementation of a max-heap data structure and the heap sort algorithm in Python, as Task 2 of the Course Project.

## Contents

- `heap.py` – `MaxHeap` class with insert, extract_max, peek, and helper methods.
- `heapsort.py` – Heap sort implementation with both recursive and iterative heapify options.
- `test_heap.py` – Comprehensive test suite covering heap operations, priority queue, top‑k elements, heap sort on various inputs, edge cases, performance measurements, and adversarial inputs.

## Requirements

- Python 3.6 or higher (no external libraries needed).

## How to Run

1. Clone the repository or download the three `.py` files into the same directory.
2. Open a terminal in that directory.
3. Run the test suite:
   ```bash
   python test_heap.py
